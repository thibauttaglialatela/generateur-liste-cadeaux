# Générateur de liste de cadeaux

Ce projet, développé avec Symfony, utilise un agent IA pour générer une liste de cadeaux personnalisée en fonction de l'âge et du sexe de la personne à qui l'utilisateur souhaite offrir des cadeaux.  

La liste générée est exportée sous la forme d'un fichier JSON.

## Technologies utilisées

### Backend
- Symfony 7.4
- PHP 8.2
- API Mistral

### Frontend
- TailwindCSS avec DaisyUI
- Asset Mapper de Symfony
